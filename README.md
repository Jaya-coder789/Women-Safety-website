 🚨 Women Safety Website
 ------------------------
📌 Introduction

The Women Safety Website is a smart safety web application developed to help women during emergency situations.
This project provides quick access to emergency support features like SOS alerts, fake calls, emergency email alerts, police help, hospital support, and live location sharing.
The main purpose of this project is to improve women’s safety using modern web technologies.

----------
🎯 Objective

The main objectives of this project are:

Provide quick emergency support
Send SOS alerts during danger situations
Share live location with emergency contacts
Help users contact police and hospitals quickly
Create awareness about women safety using technology

📊 Methodology

1️⃣ User logs into the website

2️⃣ User adds emergency email/contact

3️⃣ During emergency, user presses SOS button

4️⃣ System sends alert message with location

5️⃣ User can access police or hospital support instantly

6️⃣ Fake call feature helps users escape unsafe situations
